## Definição
É um [[Programa]] chamado "terminal emulator". É uma janela que ao abrir, podemos interagir com o [[Shell]].

**Comandos simples**
Seta para cima — navegar no histórico de comandos no terminal
Puxar textos para o terminal —  o mesmo que `Ctrl + c` e `Ctrl + v`
