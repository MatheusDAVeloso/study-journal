# Flutter

---
## Inicialização
`flutter create <app-name>` -> criar um projeto em Flutter do zero

---

>[!BUG] Erros comuns
>1. `Cannot create a project within the Flutter SDK. Target directory 'C:\Users\user\develop\flutter\agent_mdav' is within the Flutter SDK at 'C:\Users\user\develop\flutter'.`
>O problema acontece porque está tentando criar um projeto Flutter dentro do arquivo onde ele foi configurado. O projeto deve ser criado em outro caminho.
