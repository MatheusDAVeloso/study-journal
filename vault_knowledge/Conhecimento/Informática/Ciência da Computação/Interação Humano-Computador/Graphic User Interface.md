# Graphic User Interface

---
## Conceitos
- **Container** — tem como objetivo organizar e guardar widgets.
- **Widget** — é um componente qualquer na tela.
- **Event handler** — tratadores de eventos. Exemplo, ao clicar em um botão, uma tarefa é executada e essa tarefa é o **event handler**.
- **Event loop** — Um **event loop** verifica constantemente se outro evento foi adicionado. Se a hipótese for verdadeira, ele irá executar a rotina correspondente.
