# Linux

---
## Criador/Descobridor
Linus Torvalds

---
## Definição
O Linux é um sistema operacional open-source [[Unix-like]]. Na maioria dos sistema Linux, utiliza-se a CLI do [[Bash]], e para podermos interagir com ele, utilizamos um [[Terminal]].
Ele permite qualquer alteração em seus sistema, dado que Linux subentende que todos usuários sabem o que estão fazendo.

---
## Comandos
### 1.  Comandos de Limpeza
`clear`

### 2. Comandos de diretório
#### 2.1. Criação
`mkdir <directory-name>` — make directory

#### 2.2. Listagem
`ls` — list
* `-l` — long — forma detalhada
- Se um arquivo aparecer verde ou com um "`*`" significa que é executável

### 2. Comandos de filtragem
`grep '<text>' <file.txt>` — filtra pelo texto passado entre aspas simples e traz todas as linhas em que foram achadas o termo em um arquivo. Se algum comando jogar o output dele aqui, vai no lugar do `<file.txt>`
