# Shell

---
## Definição
Um [[Software]] que interpreta comandos e envia para o sistema operacional — incluindo executar scripts.

---
## Comandos de Diretório

### 1. Print
`pwd` — print work directory — mostra o caminho completo até o diretório atual

### 2. Navegação
`cd` — change directory
-  `~` ou `~<user-name>`  — home
- `..` — subir um nível — caminho relativo
-  `<folder-name>` ou `./<folder-name>` — caminho relativo
- `/<folder-name>` — caminho absoluto — começa da raiz

---
## Comandos de encadeamento

`<output-command> | <input-command>` — encadeia a saída de um comando na entrada de outro.

---
## Comando de execução de [[Software]]

`<software-name> .` — do sistema ou integrado a ele 
`./<file-name>` — criado por um usuário
