# Entity Component System

---
## Entidades
Entidades são coisas únicas no jogo e são representados por um único número inteiro. E podem ter componentes.
## Componentes
Componentes podem ser representados como um vetor de identificação de componentes.

>[!EXAMPLE] Exemplo
>Por exemplo a entidade player poderia ter os componentes Position e Health. Então o tipo da entidade seria \[Position, Health\]
