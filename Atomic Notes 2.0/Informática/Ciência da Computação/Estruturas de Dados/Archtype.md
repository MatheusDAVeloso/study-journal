# Archtype

---
## Definição
Uma [[Estrutura de dados]] que guarda todas as entidades que tem um mesmo componente. Por causa disso, todas as entidades guardadas em um Archtype podem compartilhar o mesmo ID de componente o que reduz muita sobrecarga de memória
