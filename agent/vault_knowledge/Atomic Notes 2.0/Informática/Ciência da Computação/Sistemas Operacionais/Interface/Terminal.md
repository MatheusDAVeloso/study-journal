# Terminal

---
## Definição
É um [[Software]] que se abre em forma de uma janela, podendo interagir com o [[Shell]].

---
## Comandos
Seta para cima — navegar no histórico de comandos no terminal
Puxar textos para o terminal —  o mesmo que `Ctrl + c` e `Ctrl + v`
